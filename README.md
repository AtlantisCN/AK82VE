## 灯光
### 方式一：
旋钮左右旋=灯效切换
按下旋钮=开关灯

### 方式二：
FN（字母区空格右边第2个：alt，FN，ctrl）
FN+左右上下=灯开关，灯效，亮度加减；

## 关于灯光供电

灯珠多，挺耗电，尽量插在USB 3.0接口上；


## 如有异常情况

背后小开关为灯光开关，可以强制关灯；
